Overview
This is a very simple example of mock integration with Salesforce through Mulesoft and is presented as a PoC.
Below are sample diagrams to illustrate how I usually specify the flow.
Architecture
System landscape: Salesforce REST Apex service exposed to provide CV details from database, TextLocal SMS provider service to provide notification when CV requested, Anypoint Mulesoft pltform in the middle to orchestrate requests and convert data to PDF.

Runtime: Mule 4.1.4, v0.1 Core, Java 8
Salesforce API version: 45
Java libraries used:
  • com.google.code.gson.gson
  • com.lowagie.itext
  • com.itextpdf.itextpdf
  • com.itextpdf.kernel
  • com.itextpdf.layout
